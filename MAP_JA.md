# How to use / 使い方
`C:\Users\ユーザー名\AppData\Local\DELTARUNE`  
を開き、その中にある任意の設定ファイル　　
(filech1_0が1番目、filech1_1が2番目、filech1_2が3番目のセーブスロットに対応しています。3と9はわからないです。)  
をメモ帳等で開き、下から二番目の行に任意のマップIDを入れてください。
